# 🏴‍☠️ One Piece: Grand Sea Battle (EFL Edition)

An interactive, web-based vocabulary battle game designed for young English learners (Grade 1 & 2). Students can battle each other or face off against a computer opponent to practice English vocabulary through visual recognition and audio cues.

## 🚀 Key Features
- **1 Player vs. CPU Mode:** Students can play solo against a computer opponent with selectable difficulty levels (Easy, Normal, Hard).
- **2 Player Mode:** A side-by-side battle mode for classroom competition.
- **Dynamic Word Editor:** Teachers can customize the vocabulary word bank directly in the browser.
- **QR Code Generator:** Generate a QR code in the editor to instantly share the game with students on mobile devices or tablets.
- **Voice Synthesis:** Automatically speaks the target word to assist with pronunciation and listening skills.
- **Child-Friendly Mechanics:** Incorrect answers play a "wrong" sound but do not penalize the player's health, encouraging continued play.

## 📖 How to Use
1. **Host on GitHub Pages:** Upload `index.html` and all asset files (`.gif`, `.jpg`, `.mp3`) to a GitHub repository and enable GitHub Pages in the settings.
2. **Configure Vocabulary:** Click **QUESTION EDITOR** to add your current weekly vocabulary.
3. **Generate QR:** Press **GENERATE QR** and display it on your smart board for students to scan.
4. **Choose Mode:** Students select **1 PLAYER** or **2 PLAYERS**.
5. **Start Battle:** Select a character and begin identifying images that match the spoken English word.

## 📁 Required Assets
For the game to function fully, the following files should be in the root directory:
- **Audio:** `title_music.mp3`, `hit.mp3`, `win.mp3`, `fight.mp3`, `wrong.mp3`, and character-specific lines (e.g., `luffy_line.mp3`).
- **Images/Gifs:** `luffy.gif`, `zoro.gif`, `sanji.gif`, `nami.gif`, `chopper.gif`, and your vocabulary images (e.g., `quilt.jpg`, `sun.jpg`).
